```
┌──────────────┐
│   User       │
│  (Browser)   │
└──────┬───────┘
       │
       │ 1. Navigate to /login
       ▼
┌─────────────────────────────────────┐
│  Login Page (app/(auth)/login)      │
│  - Email/Password Form              │
│  - Calls signIn() server action     │

       │
       │ 2. Submit Form → signIn()
       ▼
┌─────────────────────────────────────┐
│  Server Action (auth.actions.ts)    │
│  - Validate with Zod                │
│  - Call Supabase auth.signInWithPassword() │
│  - Fetch user profile from Prisma   │
```

## Key Data Flows

### 1. User Creation Flow
```
Admin Dashboard → createUser() action → Supabase Admin API → Prisma User Create
                                                              ↓
                                                    Role-specific profile create
                                                    (Teacher/Student/Parent)
```

### 2. Attendance Flow
```
Teacher Dashboard → markAttendance() action → Prisma Attendance Create
                                                              ↓
                                                    Update AttendanceLock
```

### 3. Assignment Flow
```
Teacher → createAssignment() → Prisma Assignment Create
Student → submitAssignment() → Prisma Submission Create
Teacher → gradeSubmission() → Prisma Submission Update
```

### 4. Exam Flow
```
Admin → createTestSession() → Prisma TestSession Create
Teacher → scheduleExam() → Prisma Exam Create
Teacher → recordResults() → Prisma ExamResult Create
```

## Technology Stack Summary

- **Framework**: Next.js 15 (App Router) + React 19
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **UI Components**: shadcn/ui (Radix UI primitives)
- **Database**: PostgreSQL
- **ORM**: Prisma
- **Authentication**: Supabase Auth
- **File Storage**: Supabase Storage
- **State Management**: Zustand
- **Forms**: React Hook Form + Zod validation
- **Charts**: Recharts
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **3D**: Three.js + React Three Fiber (for visual elements)
